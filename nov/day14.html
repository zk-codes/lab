Before Flex

layout methods we almost forgot

Flexbox changed everything. Suddenly, centering was easy. Responsive layouts became intuitive. We reached for display: flex without thinking. But what did we use before?

The Methods

CSS Grid

The newer alternative. Two-dimensional layout that handles both rows and columns.

Floats

The classic hack. Originally for wrapping text around images, repurposed for columns.

Inline-Block

Elements that flow like text but accept width and height. Watch for whitespace gaps.

Table Display

CSS table properties without HTML tables. Equal-height columns for free.
