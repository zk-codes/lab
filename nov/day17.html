Still Life

visual interest without motion

Nothing moves on this page. No fading, no sliding, no pulsing, no spinning. Every visual effect is static, frozen in place like a painting.

Animation has become the default. Buttons subtly grow on hover. Menus slide into view. Loading spinners rotate endlessly. We've trained users to expect movement.

Static Tools

Patterns create visual texture without requiring motion. This box uses repeating gradients - complex enough to hold attention, simple enough to stay still.

STATIC

Typography can play. Transforms like rotation and scale add dynamism to letterforms. The eye moves even when the elements don't.

The photograph stopped time. The painting never started it. Perhaps stillness is not absence of interest, but presence of focus.

Gradients shift without shifting.

Shadows suggest depth. Borders create rhythm. Color draws the eye. None of these need to change over time to communicate effectively.

The constraint reveals: animation is enhancement, not requirement. The page can breathe without literally breathing.
