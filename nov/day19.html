Off the Grid

layout without CSS Grid

CSS Grid revolutionized web layout. Two-dimensional control, implicit and explicit tracks, named areas - it's powerful. But it's not the only way.

Flexbox Approach

One Dimension

Flexbox excels at distributing space along a single axis. Wrap creates rows.

Flexible Items

Items grow and shrink based on available space and content.

Alignment

Centering and distribution become trivial with justify and align.

Float Approach

The Classic

Before flexbox and grid, floats built most of the web. They still work.

Clearfix Required

Floats collapse their container. The clearfix pattern solves this.

Inline-Block

Inline-block elements flow like text but accept width and height. Watch for whitespace between elements - set font-size to zero on the container.

Each layout method has its strengths. Grid isn't always better - it's just different. The constraint reminds us to choose the right tool for each job.

This page uses display: flex, float, display: inline-block, and position. No grid anywhere. The layouts work, the page responds, and sometimes the older tools fit better.
