No Margins

the other spacing tools

Margin and padding are often confused. Both create space. But margin pushes elements apart; padding expands them inward. Today, we only have padding.

The Alternatives

Padding

Internal space that expands the element.

Gap

Flex and Grid gap handles child spacing.

Borders

Transparent borders can act as spacers.

This page uses gap on the main element to space paragraphs. No margin on any child. The parent handles all vertical rhythm.

Margin collapses. Margin creates space outside. Margin is invisible but affects siblings. Padding is simpler: it only affects the element itself.

This box has no margin. Its separation from surrounding elements comes from the parent's gap property. Its internal breathing room comes from padding.

The constraint reveals how much we rely on margin by default. But modern layout tools - flexbox gap, grid gap - handle spacing between elements cleanly. Margin becomes optional.
