Positioned Otherwise

layouts without position: absolute

The position: absolute property removes elements from document flow and places them precisely. It's powerful, but it creates fragility. Today we build without it.

The Remaining Options

Static

The default. Elements flow normally, one after another.

Relative

Offsets from normal position. Space is preserved in the flow.

Sticky

Hybrid of relative and fixed. This header demonstrates it.

Fixed

Positioned relative to viewport. See the note in the corner.

Absolute positioning solves problems quickly. But it creates new ones: elements that don't respond to their siblings, layouts that break on resize. The alternatives are more resilient.

Grid and flexbox reduce the need for absolute positioning. Sticky headers replace absolutely-positioned fixed elements. Transform and margin create offsets. The modern layout toolkit makes absolute positioning optional.
