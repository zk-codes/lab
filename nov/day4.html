Self-Contained

everything in one place

This entire page exists in a single HTML file. No external stylesheets. No linked CSS files. Every rule lives here, either in the <style> tag above or inline on the elements themselves.

The benefits of self-containment:

One file to share, one file to save

No broken links to external resources

Works offline, works forever

The document carries its own presentation

There was a time when this was the only way. Before CSS existed, styles were baked into HTML attributes. <font color="red" size="3"> ruled the web.

"The best code is sometimes the code that doesn't require a build step, a bundler, or a network request."

This constraint reminds us that simplicity has value. A single file is portable. It's archivable. It will render in any browser, on any device, at any time in the future. The web was built on documents, and documents can be self-sufficient.

Save this page. Disconnect from the internet. Open it again. It will look exactly the same. That's the promise of the contained document.
