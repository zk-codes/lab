A Eulogy For <blink>

⚠ THIS PAGE USES DEPRECATED ELEMENTS ⚠

Some tags were too beautiful for this world. They burned bright, annoyed everyone, and were quietly removed from the specification while we weren't looking.

★ WELCOME TO THE MUSEUM OF DEPRECATED HTML ★ PLEASE DO NOT FEED THE ELEMENTS ★ PHOTOGRAPHY IS ENCOURAGED ★

We built websites with <center> and <font face="Comic Sans">. We nested tables inside tables inside tables. It was chaos. It was beautiful.

// Element Graveyard

  • <blink> 1994-2013 — Beloved by Geocities, hated by everyone else
  • <marquee> 1996-∞ — Still works. Still shouldn't.
  • <center> 1995-2014 — Replaced by flexbox (eventually)
  • <font> 1995-2014 — 15 attributes, zero semantics
  • <frame> 1996-2014 — Navigation was never the same

Progress requires sacrifice. But sometimes, late at night, I still miss the chaos. The web was weird. The web was free. Now it's just...optimal.

This page is a memorial. And a warning. Everything you love will eventually be deprecated.
